More information

The Fantom premium is a new version of the Fantom human-readable address. In this idea, we tried to use .FTM and .FTMP.COM domain address. As you know, human-readable addresses are a short version of the commonly used address like this: 0xdadd00000. We want to convert .FTM domain to > .FTMP.COM domain and making that usable on the world wide web as a common domain which supported by all the browsers. We can test this feature on real word if we have your support.

Used technologies:

Fantom network

We are using fantom network for hosting our solidity codes. low fee and stable network.

Rave names

An auxiliary technology for showing up the chrome extension functionality.

Extension

A Google Chrome extension for calling solidity functions and converting those data's into user-friendly content.

Donation page

In this section user will be eligible to use .FTM / .FTMP.COM domains on page and without any limitation which is applied by wallets.

Metamask/wallet connect

Converting .FTM / .FTMP.COM domains into a usable address(hexadecimal 0x000 format) by deep linking for donation and advertisement purpose.

Reward system

This section isn't applied in the beta version. but we have a protocol which is rewarding does who used the .FTMP.COM domains.


There's three main purposes that can be ideal for us:
1. Using .FTMP.COM will make us eligible to earn rank 1 on the search engine query searches.
2. helping community to use this new version of HRAs in their social media and making FTMP as a peer to peer advertisement tool.
If somebody use .FTMP.COM domain (Example_user.FTMP.COM) in its social media profile will be eligible to receive community donation in the easiest way (replacement of classic way) and it is either a social media back link which help .FTMP.COM earn better ranking in searches.
3. Wallets like metamask doesn't support .FTM / .FTM.COM very well. Our Chrome extension will solve this problem too.
We focused on its SEO (Search engine optimization), and maybe it becomes trend in social media and earn more attentions to projects as a low-cost (0$) advertisement.
